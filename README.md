# Web Development Lab 1

## Overview
This is the first web development lab of the semester. The project focuses on learning and practicing basic HTML structure and workflow using development tools and version control systems.

## Features
- Basic HTML file creation and structure
- Introduction to web page layout
- Practice with file organization and editing

## Tools Used
- HTML5
- Visual Studio Code
- GitHub (Web and Desktop)
- Basic Git version control workflow

## Learning Outcomes
- Understood the structure of a basic HTML document
- Learned how to use GitHub for version control and project storage
- Gained experience using Visual Studio Code for web development
- Practiced committing and managing project files through GitHub

## Purpose
The purpose of this lab was to introduce foundational web development concepts and tools, including HTML and basic version control using GitHub.
